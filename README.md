## 🌦 About climafy

climafy is a weather forecast app made to show clear weather details right on your Windows PC. It uses live weather data to give you daily and hourly forecasts. This app works in your browser using standard web technologies: HTML, CSS, and JavaScript. You never need to worry about installing heavy software or extra tools. Just open it, and see the weather.

## 🧰 Technical Details (For Interested Users)

climafy uses the Fetch API to get data from a weather API. It handles errors gracefully, showing messages if data cannot be loaded. The layout uses CSS Grid and Flexbox for a responsive interface. All code is in Vanilla JavaScript and runs client-side, meaning no data is sent to servers from your computer except the publicly available weather API requests.
